Automated Testing with Playwright

## Project Overview

This project contains automated tests for the IOBM Learning Management System (LMS) using Playwright. The tests cover:

- Valid login
- Invalid login
- Empty fields validation

The project demonstrates how to create reliable end-to-end tests for web applications, run tests on multiple browsers, and record test execution for debugging.

## Features

- End-to-end automated testing using **Playwright**  
- Tests for valid, invalid, and empty login scenarios  
- Multi-browser support: **Chrome** and **Microsoft Edge**  
- Automatic video recording for all test executions  
- HTML reports for easy test result review

---

## Tech Stack

- **JavaScript (Node.js)**  
- **Playwright Test Runner**  
- **HTML Reporter**  
- **Browsers:** Chromium-based Chrome and Microsoft Edge
